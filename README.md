   Why I started WPDK?
=========================

I'm looking for a remote position such as Senior PHP Developer.
What I depend on that job hunting process is the motto
"Write code which run faster with less memory but still be easy
to understand". I'm building the demo which can use working
experience that I gained from 15 years working in software
outsourcing industry.

A romantic but something stupid idea comming is building a wordpress
plugin / theme which can help system running with at least 1 billion
posts and can be deployed in free hosting like InfinityFree.com
( 50 GB disk space ). I soon recognize that it's the stupid idea
because that hosting only allow 100,000 requests to database each hour.
The more is storing 1 billion normal posts requires at least 2TB disks
(that is based on my calculation).

When I see job posting from Automattic for Experienced Software Engineer,
I think this idea is the best personal project to build demo for
demonstrating my skills and experience which show that I have ability
to finish projects assigned by Automattic. Some noticable points are:
+ Project will be built in 1 month
+ Project will be done in free host (InfinityFree) for small data to test
availability. It then will be deployed on hosting with "iFastNet
Ultimate Premium" package ($16 / 2 months) for testing. If time remains
and I have enough money, it will be deployed to DigitalOcean, AWS,
Google Cloud to benchmark in order to find out which hosts gain
maximum profit in each use case.
+ Project will be coded following Wordpress Coding Standards and will
be optimized for:
  o being easy to integrate to Wordpress.com platform (Wordpress.com will
    be simulated based on my knowing about the platform and for demonstrating)
  o easy and safely to remove from Wordpress.com platform 
  o reduce maximum operation cost and provide new service for wordpress.com
    with maximum profit (the project may be not accomplish this goal but
    it shows that I know how to do, be interested to do, be ready to learn
    and be able to accomplish with the helps from other experienced software
    engineers). 
+ Project will be open sourced to learn and practise the motto of Automattic,
in other says, I will do this project by learning and practising the way to do
open source of Wordpress.
